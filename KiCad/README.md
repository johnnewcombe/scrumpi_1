# KiCad Files

This directory contains the KiCad project and Gerber files for a prototype Scrumpi 1 board.

Several replica boards were created following extensive discussions by members of the 
Vintage Radio Repair and Restoration Forum (see https://www.vintage-radio.net/forum/showthread.php?t=166274). 
The KiCad files here are based on photographs of the latest of these boards.

For full details of the Scrumpi and its preservation please see http://www.scrumpi.uk/.
